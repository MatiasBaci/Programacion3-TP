Trabajo Práctico Grupal para la materia de Programación 3, cátedra de Cano/Brasburg/Raik, 2do cuatrimestre 2023.

La consigna fue programar el combate de los videojuegos Pokémon usando programación orientada a objetos, patrones de diseño, eventos, persistencia, unit tests, arquitectura MVC, JavaFX.

Versión de Java: 20.0.2

Para correr el juego abrir una consola en la carpeta \Programacion3-TP y ejecutar el comando:

java -jar --enable-preview out\artifacts\TpJava_jar\TpJava.jar
